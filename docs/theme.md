# テーマ — ブラッシュ・ノワール

サイトの配色は **ブラッシュ・ノワール** に固定しています。

## 設計

| 役割 | 内容 |
| --- | --- |
| ベース（60%） | 冷たい薄ライラック |
| 構造（30%） | 藤系のカード・ボーダー |
| CTA（10%） | ワイン系（`primary`） |
| 本文 | 暖色寄りグレー（紫がかった文字にしない） |

定義: `src/lib/theme/colors.css`（`:root` の CSS 変数）

## トークン

shadcn / Tailwind と同じ変数名（`--background`, `--primary` など）を使います。装飾用に `--brand-gradient-*` と `--brand-glow` もあります。
